# 2026 新锐期刊分区速查辅助工具

> 基于 [XR-Scholar](https://www.xr-scholar.com/) 2026 年新锐期刊分区表，提供大类/小类学科分区及 Web of Science 数据库的快速查询辅助。

## 在线使用

👉 **<https://hylouis.github.io/xr-journal-query/>**

无需安装，打开即用。也可下载 `index.html` 文件到本地，双击浏览器打开离线使用。

## 功能

- 覆盖 **21 个学科大类**、**22,257 本期刊**
- 输入期刊名称即时搜索，结果实时筛选
- 按学科大类/小类、分区（1-4 区）、Top 期刊筛选
- 大类与小类级联联动筛选
- 支持按 Web of Science 数据库收录状态筛选
- 每条结果可跳转至 XR-Scholar 官网查看详情
- 单 HTML 文件，约 1MB，支持离线使用
- 适配手机浏览器

## 数据说明

- **数据来源**：[XR-Scholar 新锐期刊分区表](https://www.xr-scholar.com/Category/ZKY)
- **数据年份**：2026 年
- **收录范围**：大类学科分区（21 个学科方向）+ 小类学科分区
- **展示字段**：期刊名称、大类学科、小类学科、分区（1-4 区）、是否 Top、Web of Science 收录状态
- ISSN、EISSN、影响因子等详细信息请前往 [XR-Scholar 官网](https://www.xr-scholar.com/) 查询
